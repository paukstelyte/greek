Publish a new
game in minutes

  1. 1.Build a new game as an .html file — ask Claude to make one!
  2. 2.In your GitHub repo click Add file → Upload files.
  3. 3.Upload the new .html file to the root folder.
  4. 4.Edit index.html ✏️ and copy any existing <a class="card"> block.
  5. 5.Update the href, photo URL, title and description.
  6. 6.Click Commit changes — your page is live in ~60 seconds.
